- Shows where the waves come from (NW, W, SW).

### 4. Wind Speed & Direction
- Wind can make waves clean or messy.
- Best: **Light offshore wind** (E or NE).
- Worst: **Strong onshore wind** turning waves choppy.

- Ocean Beach changes a lot with tides.
- Mid tide often best for shape.
